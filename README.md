# Blog App

## Built With

- Ruby
- Ruby on Rails

## Setup

```
 git clone https://github.com/MICHAELMUNAVU83/blog.git
 cd blog
```
## Install gems

```
 bundle install
```
## Testing
```
Tests are present in the spec folder.

run rspec --format documentation spec/ to check all test cases
```
